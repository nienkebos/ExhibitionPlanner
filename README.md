# Exhibition Planner
This is an app to store information about planned exhibitions. This exhibition planner is build with React on a Rails backend.


## Database
1 Exhibition
  * Title:string
  * Date:string
  * Image:string
  * Description:text
2 Artwork
  * Artist:string
  * Title:string
  * Date:string
  * Technique:string
  * Image:string
  * Description:text
  <!-- * Later on I need to include audio -->
